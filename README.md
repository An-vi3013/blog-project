# Blog Application using Flask

## Description
This is a simple blog web application built using Flask. It allows users to navigate between pages like home, login, and create post.

## Features
- Home page
- Login page
- Create post page
- Simple UI with CSS

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000
